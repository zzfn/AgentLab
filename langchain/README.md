# LangChain 演示

使用 LangChain + OpenAI 兼容 API 的演示代码。

## 演示列表

| 文件                   | 内容                                                  |
| ---------------------- | ----------------------------------------------------- |
| `01_langchain_demo.py` | LangChain LCEL 综合演示（对话、模板、链、流式输出等） |

## 环境配置

```bash
export OPENAI_API_KEY="your_key"
export OPENAI_API_BASE="https://your-api-endpoint.com"
```

## 运行

```bash
uv run python langchain/01_langchain_demo.py
```
